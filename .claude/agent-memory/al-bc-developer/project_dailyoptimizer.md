**General technique — decompiling a base-app table when `al_symbolsearch` can't answer the question (established 2026-07-09):** `al_symbolsearch` only exposes field signatures (e.g. `"Recurring System": Option`), never Option `OptionMembers`/values, trigger bodies, or other implementation detail for base-app (non-project) objects.
